The module is used for computing a model for cosmic birefringence where an energy transfer from matter to ALPs happens during some spefici period after the recombinaion. 

To run this module, you need to install
- pyrefCLASS (https://github.com/deer-astron/pyrefCLASS)
- birefCLASS (https://github.com/toshiyan/ongoing/tree/master/rotation/code/birefclass_lens)

Please also create a directory to save output files and change the path to modules in the files.
